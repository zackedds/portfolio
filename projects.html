    SQL Databasing

    The SQL query of the USURDB (United States Utility Rate Database) was not returning the desired electrical rate information. The query was modified and a Python script was used to identify missing rates. The query was corrected but further analysis found the database itself contained incorrect information. The USURDB was notified and correct rate information was published.

    Advanced Excel Dev

    Developed algorithm to calculate EV charging cost based on varying electricy demand and charger current draw. Users selected their local electrical rate, EV battery capacity, and charger for to compute their charging cost estimate. A data validation drop down was developed to build a dynamic range of electrical rates based on the state and type input.
